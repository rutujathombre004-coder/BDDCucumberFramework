# BDDCucumberFramework

This project was created during my automation testing training. 
It is a practice project to understand Selenium automation with Cucumber BDD.

## Tools used:
- Java
- Selenium WebDriver
- Cucumber BDD
- TestNG
- Maven
- Eclipse IDE

## Application tested:
- Public e-commerce website (Flipkart) for practice purposes only

## Scenarios automated:
- Login with valid credentials
- Login with invalid credentials
- Add product to cart
